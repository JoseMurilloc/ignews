## 🏁 Starting ##

```bash
# Clone this project
$ git clone https://github.com/JoseMurilloc/ignews

# Access
$ cd ignews

# Install dependencies
$ yarn

# Run the project
$ yarn dev

# The server will initialize in the <http://localhost:3000>
```
